# Physical Information

A comparative thermodynamic and machine learning framework linking DNA, brain memory, and digital computation through the shared principle that stored information is physically embodied, vulnerable to entropy, and maintained by energy-dependent processes.

## Objective
to develop and test a comparative framework showing that DNA, brain memory, and digital computation are distinct forms of physically embodied information whose preservation is governed by thermodynamic constraint and energy-dependent maintenance.

## Overview
This repository contains the conceptual manuscript, literature-anchored synthetic benchmark notebook, and supporting materials for a cross-domain study of information storage in three physically distinct systems:

- **DNA**, where information is encoded in nucleotide sequence and chromatin organization.
- **Brain memory**, where information is distributed across synaptic strength, receptor states, structural plasticity, and network dynamics.
- **Digital computation**, where information is encoded in physically implemented binary states.

The project asks a unifying question: can these very different systems be compared within a shared thermodynamic framework in which information is treated as a physical pattern instantiated in a physical substrate, exposed to entropy, and preserved by energy-dependent maintenance?

## Central idea

The repository is organized around the following principle:

stored information в†’ physical embodiment в†’ susceptibility to entropy в†’ need for energy

## Notebook summary

The notebook generates three synthetic datasets inspired by peer-reviewed journal literature:

1. **DNA dataset**
   - lesion burden
   - oxidative fraction
   - repair efficiency
   - chromatin remodeling support
   - replication stress
   - antioxidant support
   - cell-cycle pressure
   - outcome: **DNA information fidelity**

2. **Brain dataset**
   - glucose utilization
   - glycogen support
   - protein synthesis capacity
   - synaptic plasticity
   - sleep-related consolidation support
   - oxidative stress
   - network noise
   - outcome: **memory retention score**

3. **Computer-bit dataset**
   - temperature
   - energy barrier relative to the Landauer bound
   - hardware noise
   - error-correction strength
   - practical reset-energy burden
   - outcome: **bit reliability score**

These datasets are then analyzed using descriptive statistics, visualization, regression, classification, permutation importance, PCA, t-SNE, K-means clustering, and cross-system comparative modeling.

## What this repository is not

This repository is **not** a substitute for direct experimental measurement. The notebook uses **synthetic data**, not raw empirical datasets. The numerical ranges are literature-anchored and biologically or physically plausible, but the outputs should be interpreted as a **proof-of-principle computational framework**, not as a direct discovery study.

## Main message

DNA, brain memory, and computer bits are mechanistically different, but they all illustrate the same deep rule:

- stored information must be physically instantiated,
- physical instantiation is vulnerable to disorder,
- preserving meaningful state distinctions requires energy.
